# Exp4 – Student Bank (Flask)

A simple web app simulating a student bank login built with Python and Flask.

## Prerequisites

- Python 3.7 or higher
- Flask

## Setup

Install the required dependency:

```bash
pip install flask
```

## How to Run

From inside the `exp4/` folder, run:

```bash
python app.py
```

Then open your browser and visit:

```
http://localhost:5000
```
