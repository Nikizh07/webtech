# Exp7 – Real-Time Chat (Node.js + Socket.io)

A real-time chat app with a bot that echoes your messages, built with Node.js, Express, and Socket.io.

## Prerequisites

- Node.js 14 or higher
- npm

## Setup

Install dependencies from inside the `exp7/` folder:

```bash
npm install
```

## How to Run

```bash
npm start
```

The server starts on **http://localhost:5000**.

Open your browser and visit:

```
http://localhost:5000
```

Type a message and send it — the bot will reply automatically within 1 second.
